⊙族谱网：把族谱从纸质转移到网络上，实现数字化保存。

个人中心：宗豆(ZD)、经验值、相册、日志、故事、收藏、充值、资助、
    个人资料：
        基本：生日年龄(阳历农历)、属相(八卦命相)、星座()、英文名、昵称外号(多个)、
        联系方式：手机电话/QQ/微信/支付宝/邮箱/住址/工作地/老家地址/自定义text(0)、
        教育及工作：毕业院校/学历、公司单位/职业/薪资/待遇/公司性质/人员、
        家庭(最近一级：妻子或丈夫/父母/儿女)、
        我是谁：性格、兴趣爱好(唱歌/书法)、座右铭、故事、
        其它(聊友、自定义)
